C#.NET - MSFS2020 Pilot Logbook

For the flight simulation game mania, The MSFS2020 have uncustomized pilot logbook. This program is helping your recording the individual flight logs and calculates of total flight time, total takeoffs and landings day/night. The application is created by C# WinForms with fully MySql Database for manipulation of the data and using the closedxml framework for exporting the logs into .xlsx files.



C#.NET - Lab Booking Windows Application

This program is designed for my present work and easy to use. They have basic function including date, name, room and purpose to use. In case of unavailable room, you will see the labeled (top-right) is red and shows the information. In addition, The program is easy adaptation to another application such as meeting room booking, restaurant system, calendar appointment system, etc.



C#.NET - QR Code Generator

A small windows form application program creating by C# with ZenBarcode framework with MySql Database to stored the history. This program is designed for easy to use, fast, compacted, very usefully for everyone and in daily life.
